# Changelog

## 0.1.0 — 2026-05-14

### Added

- **`@tokidevs/adachi`** npm package: ESM build, TypeScript declarations, pre-built `styles.css` (Tailwind v4 + design tokens).
- **Components:** `Button`, `Card`, `Input`, `Modal` (focus trap + scroll lock), `Dropdown` (keyboard navigation).
- **Theming:** `ThemeProvider`, `useTheme`, four presets (`light`, `dark`, `neon`, `glass`), configurable `storageKey`.
- **Monorepo:** `apps/web` showcase consuming the workspace package.
- **CI:** GitHub Actions (lint, typecheck, test, build).
- **Docs:** library README, CONTRIBUTING, MIT LICENSE.
